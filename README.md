# MONDAY CUP Vite app

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this folder to GitHub.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repo.
4. Framework preset should auto-detect as **Vite**.
5. Build command: `npm run build`.
6. Output directory: `dist`.
